URL(UNIFORM RESOURCE LOCATOR):

URL (uniform resource locator) it is the address used to access the resource address. The structure of URL is divided into three classes they are:

1.protocol

2.Domain

3.path

1.Protocol

It is the set guidelines or rules helps us to display the data over the internet. There are different types in protocol mainly:

1.http

2.https

2.Domain

Domain make it easy for users to access websites without needing to remember IP addresses.

EX:www.amazon.com

3.Path

The Path specifies the location of the resource on the server.

EX:login? In www.amazon.com/login?

WATERFALL METHODOLOGY

1.The Waterfall methodology is a traditional, linear project management approach often used in software development and other industries.

2.It is characterized by a sequential design process, where each phase of the project must be completed before moving on to the next.

3.The method is structured and often used when the project requirements are well understood and unlikely to change significantly during development.

SDLC (SOFTWARE DEVELOPMENT LIFE CYCLE)

The Software Development Life Cycle (SDLC) is a process that consists of several stages for developing software:

  1. REQUIREMENTS ANALYSIS
  2. PLANNING
  3. SYSTEM DESIGN
  4. IMPLEMENTATION
  5. TESTING
  6. DEPLOYMENT
  7. MAINTAINANCE
